# Cube-World-Commands-Mod
For use with the Cube World Mod Launcher: https://github.com/ChrisMiuchiz/Cube-World-Mod-Launcher

## Commands:
  #### Display world coordinates
  
    /coords
    
 ---
 
 #### Teleport in terms of map coordinates

    /tp <x> <y>
 
 ---
 
 #### Set time
 
    /settime <hour>:<minute>
